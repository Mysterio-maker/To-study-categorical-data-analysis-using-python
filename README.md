1. Aim
The aim of this experiment is to study and understand categorical data analysis using Python. The goal is to analyze non-numerical data, identify patterns, and extract meaningful insights using various data analysis techniques.

2. Objective
To understand the concept of categorical data and its types. To analyze categorical variables using frequency counts and distributions. To identify relationships between different categories using cross-tabulation. To perform operations such as filtering, grouping, and sorting on categorical data. To interpret real-world datasets using categorical analysis techniques.

3. Concepts Used
3.1 Categorical Data Categorical data represents data that can be divided into groups or categories rather than measured numerically.

3.2 Types of Categorical Data

Nominal Data
Categories without any specific order. Example: Gender, Payment Method, Product Category. 2. Ordinal Data

Categories with a meaningful order. Example: Grades (A, B, C), Income Levels. 3.3 Frequency Count Frequency count refers to counting how many times each category appears in the dataset. It helps in understanding the distribution of data.

3.4 Unique Values Unique values represent all the distinct categories present in a dataset.

3.5 Cross Tabulation Cross tabulation is used to analyze the relationship between two categorical variables by creating a table of frequencies.

3.6 Percentage Distribution Percentage distribution shows the proportion of each category in terms of percentage. It helps in understanding the relative importance of each category.

3.7 Filtering Data Filtering is used to extract specific data from the dataset based on certain conditions.

3.8 Grouping Data Grouping involves combining data based on one or more categories to analyze patterns and relationships.

3.9 Sorting Data Sorting arranges data in a logical order (alphabetical or numerical) to make analysis easier.

4. Theory
4.1 Introduction to Categorical Data Analysis Categorical data analysis is an important part of exploratory data analysis. Unlike numerical data, categorical data represents qualities or characteristics. It is widely used in real-world applications such as business analytics, customer segmentation, and decision-making.
